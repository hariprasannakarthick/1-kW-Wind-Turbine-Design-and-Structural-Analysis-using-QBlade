# 1 kW Wind Turbine Design and Structural Analysis using QBlade

## 📌 Overview
This project focuses on the design and simulation of a 1 kW horizontal-axis wind turbine using QBlade. It covers aerodynamic performance using Blade Element Momentum (BEM) theory and structural validation using Q-FEM.

## ⚙️ Features
- Airfoil selection and blade design
- Rotor performance analysis using BEM
- Wind field simulation
- Structural analysis using Q-FEM
- Blade deflection and stress evaluation

## 🛠️ Tools Used
- QBlade (for design, BEM, and Q-FEM analysis)
- CAD tools (optional, if you used any)

## 📂 Project Workflow
1. Airfoil selection
2. Blade geometry design
3. Rotor setup and BEM simulation
4. Wind field configuration
5. Structural model creation
6. Q-FEM analysis for stress and deformation

## 📊 Key Parameters
- Rated Power: 1 kW
- Turbine Type: Horizontal Axis Wind Turbine (HAWT)
- Analysis Methods: BEM, Q-FEM
- Wind Profile: Turbulent (TURBSIM)

## 🚀 How to Use
1. Open QBlade
2. Load the project files from this repository
3. Run BEM simulation for aerodynamic performance
4. Create/verify blade structure model
5. Run Q-FEM for structural analysis
